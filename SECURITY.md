# Security

- **Never** commit `LICENSE_KEY`, RSA **private** keys, or `LICENSE_SIGNING_SECRET` to this repository.
- License keys are **entitlements**; treat them like passwords.
- Report security issues to the repository owner according to the upstream [rag-k8s-llm](https://github.com/sandeep27choudhary/rag-k8s-llm) security policy if one exists.
